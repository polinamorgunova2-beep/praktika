# Учёт личных финансов

Моя итоговая работа по C#. Программа, где можно записывать доходы и расходы и видеть, сколько денег осталось.

## Что тут можно делать

- добавлять доходы и расходы
- смотреть баланс по каждому счёту
- менять и удалять записи
- отменить действие назад (Ctrl+Z) и вернуть обратно (Ctrl+Y)
- смотреть операции за нужный период
- поставить лимит на категорию — программа предупредит, если потратил больше
- смотреть графики
- сделать отчёт и сохранить его в PDF
- кнопка «Заполнить примерами» сама накидает данные, чтобы было что посмотреть
- два входа: админ и гость

Всё сохраняется в файлы, так что после закрытия ничего не пропадает.

## Как запустить

1. Открыть `Учёт личных финансов.sln` в Visual Studio.
2. Собрать: Сборка → Пересобрать решение.
3. Нажать F5.

Нужен .NET Framework 4.7.2.

## Вход

Админ — пароль `admin`, может всё. Гость — без пароля, только смотреть.

## Где данные

Лежат в папке `%AppData%\FinanceTracker`. Если их нет, программа сама создаст примеры при запуске.

## Тесты

Лежат в папке `Tests`, всего 16. Сначала собрать проект, потом Тест → Обозреватель тестов → Запустить все.

## Папки

- Models — данные (операция, счёт, категория, бюджет)
- Controllers — вся логика и расчёты
- Services — сохранение и отчёты
- Forms — окна программы
- Tests — тесты
- docs — диаграмма классов

Диаграмму классов можно открыть прямо в программе кнопкой «Диаграмма классов».
