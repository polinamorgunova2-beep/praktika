## Требования к приложению

### Функциональные
- Учёт доходов и расходов: дата, сумма, категория, счёт, комментарий.
- Категории доходов/расходов; несколько счетов с пересчётом баланса.
- Добавление, редактирование, удаление операций; фильтр по периоду.
- Лимиты по категориям с предупреждением о превышении.
- Статистика (графики), отчёт в PDF.
- Вход с ролями: администратор и гость (только просмотр).

### Нефункциональные
- Хранение данных в JSON (`%AppData%\FinanceTracker`).
- Отмена/повтор действий; разделение логики и интерфейса.
- .NET Framework 4.7.2, WinForms.
